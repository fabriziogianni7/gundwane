# Gundwane Landing Page

Marketing site for [Gundwane](https://t.me/gundwane_bot) — the non-custodial DeFi agent for Telegram.

## Quick Start

```bash
npm install
npm run dev      # http://localhost:3001
```

## Stack

| Layer     | Tech                        |
|-----------|-----------------------------|
| Framework | Next.js 16 (App Router)     |
| UI        | React 19, Tailwind CSS 4    |
| Icons     | Lucide React                |
| Language  | TypeScript 5                |

## Pages

- **`/`** — Home with hero, features grid, how-it-works, pricing tiers, and CTA
- **`/about`** — Mission, technology deep-dive, values, tech stack badges

## Deployment

Works out of the box on Vercel, Netlify, or Cloudflare Pages. No environment variables needed.
